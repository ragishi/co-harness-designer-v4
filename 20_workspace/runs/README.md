# runs/ — 実行ログ / 1 回限りの作業

## 役割

V4 の設計プロセスで発生した **1 回限りの実行・バッチ処理・移行作業** のログを置く場所。
繰り返さない、または繰り返す必要がない作業の記録を残す。
「いつ・何を・なぜ実行したか」を追跡できるようにし、後から同じ作業をやり直す場合の参照元となる。

**planned stub であり、後段で本格実装する。住所と役割の確保。**
